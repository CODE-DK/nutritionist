# 🥗 Персональный Диетолог - AI-powered Nutrition App

## Что это?

**Персональный Диетолог** - мобильное приложение (iOS/Android) с AI-помощником для управления питанием и достижения целей по весу.

**Killer Feature:** 📸 **Фото-распознавание еды** - сфотографируй блюдо, AI автоматически определит калории и БЖУ за 3 секунды.

**Цель:** Сделать учет питания простым и быстрым. Вместо 10+ кликов на ввод данных - один клик на фото.

---

## Ключевые функции

### 🤖 AI Диетолог
- Чат с GPT-4 для консультаций по питанию
- Персональные рекомендации на основе целей
- Расчет калорий и составление планов питания

### 📸 Фото-распознавание (Killer Feature)
- Сфотографируй еду → получи калории и БЖУ
- Claude Vision API с точностью 70%+
- Автозаполнение дневника питания

### 📊 Дневник питания
- Учет приемов пищи (завтрак, обед, ужин, перекус)
- Автоматический подсчет калорий и макронутриентов
- Прогресс к целевому весу

### 👤 Персонализация
- Onboarding анкета (рост, вес, возраст, цель)
- Автоматический расчет BMR/TDEE
- Целевая норма калорий по формуле Mifflin-St Jeor

## Технологический стек

**Frontend:**
- React Native + Expo SDK 50
- TypeScript, React Navigation
- Локализация (ru/en)

**Backend:**
- Supabase (PostgreSQL + Auth + Edge Functions)
- OpenAI GPT-4o-mini для чата
- Anthropic Claude 3.5 Sonnet для фото

**Платежи:**
- Stripe (планируется в v2)

## Бизнес-модель

**Free tier:**
- 10 AI запросов/день
- 5 фото-распознаваний/день
- Базовая статистика

**Premium ($4.99/мес):**
- Безлимит AI чата
- Безлимит фото-распознавания
- Расширенная аналитика
- Экспорт данных

**Target:** 2-5% конверсия Free → Premium
